THE OFFICE · SETUP

The inbox isn't connected yet. One-time setup (~5 minutes, free):

  1. Go to console.firebase.google.com → Add project (call it trufoe-hub, Analytics off).
  2. Build → Firestore Database → Create (production mode).
  3. Build → Authentication → Sign-in method → enable Email/Password, then Users → Add user (Foe's login for this page).
  4. Project settings → Your apps → Web app (</>) → copy the firebaseConfig values into js/site-config.js.
  5. Firestore → Rules → paste the rules from firestore.rules in the repo → Publish.

Push the config change and this page becomes the live inbox.
