# TRU FOE (IV) — Tha Solid One · Official HUB

Cinematic single-page hub for Chicago battle rapper & recording artist **Tru Foe (4)**.
Blends every channel — Instagram, YouTube, Bandcamp, X — into one solid front.

**Live:** https://antlo314.github.io/true4/

## Stack
- Vanilla HTML/CSS/JS — no build step
- [GSAP 3 + ScrollTrigger](https://gsap.com) — reveals, counters, marquee, magnetic buttons
- [Three.js](https://threejs.org) — metallic 3D "IV" hero emblem
- Design system derived from the *Street Niggaz Don't Do Brunch* album art
  (noir · dusty rose · champagne gold · film grain)

## Run locally
```bash
python -m http.server 5188
# → http://localhost:5188
```

*Only the solid prevail.*
